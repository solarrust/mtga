When I should use tables?

Table layout can be used to represent tabular relationships between data.

Tabular data

  1. The data can be presented as rows and columns.
  2. Each row has the same columns as the other rows, in the same order.
  3. A given column has the same type, but different columns can have different types.
  4. The rows are usually in some particular order.
