Когда я должен использовать таблицы?

Таблицы может использоваться для представления табличных отношений между данными.

Табличные данные

  1. Данные состоят из строк и колонок.
  2. Каждый ряд содержит те же колонки в том же порядке, что и остальные ряды.
  3. В одной колонке представлены данные одного типа, но в разных колонках могут быть разные типы данных.
  4. Ряды обычно расположены в каком-то определённом порядке.

Клавиши VO

Tab для навигации по интерактивным элементам.

Клавиши VO + ← ↑ ↓ → для навигации по простому тексту или таблицам, расположенным на странице.

Клавиши VO по умолчанию равны Ctrl + Option.

Добавляйте role

        <table>
          <thead role="rowgroup">
            <tr role="row">
              <th role="columnheader">Имя</th>
              ...
          </thead>
        </table>

Атрибут role 🦸‍♂️

  • <table> — table
  • <thead> / <tbody> / <tfoot> — rowgroup
  • <tr> — row
  • <td> — cell
  • Заголовки сверху — columnheader для каждого
  • Заголовки сбоку — rowheader для каждого
